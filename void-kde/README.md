## Paquetes utilizados en la instalación de Void Linux junto al DE KDE Plasma.

	1) kde5
	2) kde5-baseapps
	3) ark
	4) spectacle
	5) sddm
	6) kdialog
	7) kwrite
	8) kdegraphics-thumbnailers (opcional)
